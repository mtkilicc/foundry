# Example: sample project (quality bar + fast-path walkthrough)

Fictional Django + React + Docker wholesale price-tracking app. These files
are **illustrative only** — they show the quality and shape Foundry generators
target, not a real product or private codebase.

Purpose: every generator imitates the **quality** of these files, not just
the templates' shape. Read the matching example before writing:

| Writing a... | Imitate |
|---|---|
| domain doc page | `docs/backend/domain/price-tracking.md` |
| L2 orchestrator | `skills/backend/SKILL.md` |
| leaf specialist | `skills/backend/price-tracking/SKILL.md` |
| rule | `rules/frontend-design-system.md` |
| queue event | `.foundry/queue.json` |
| small job end-to-end | `walkthrough-small-job.md` |

Rendered here in the claude-code profile dialect; the adapter re-renders
for other tools.
